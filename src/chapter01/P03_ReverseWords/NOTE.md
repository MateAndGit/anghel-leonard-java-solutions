# [Java] Strings 클래스 정밀 분석 및 실무 활용 가이드

제시된 `Strings` 클래스는 문자열 조작, 특히 '단어 단위 뒤집기'와 '전체 뒤집기'를 다루는 유틸리티 클래스입니다. 각 메서드의 내부 동작과 현업에서의 쓰임새를 정리했습니다.

---

## 1. reverseWordsV1: 반복문 기반 프로세스
이 메서드는 가장 기초적인 알고리즘 구현 방식을 보여줍니다.

### [코드 분석]
- **핵심 로직**: `split(" ")`으로 단어를 나누고, 이중 `for`문을 사용하여 각 단어의 끝 문자부터 하나씩 `StringBuilder`에 추가합니다.
- **특징**: 외부 라이브러리나 최신 Java 기능을 사용하지 않는 순수 알고리즘 중심 코드입니다.

### [실무 활용 사례]
- **알고리즘 인터뷰(Coding Test)**: 메모리 할당과 반복문의 시간 복잡도를 설명해야 하는 면용 코드로 자주 등장합니다.
- **레거시 시스템 유지보수**: Java 8 미만 버전(Android 구버전 등)의 환경에서 스트림을 사용할 수 없을 때 구현하는 표준적인 방식입니다.

---

## 2. reverseWordsV2: Stream API & 정규식
현대 Java 개발에서 가장 권장되는(Best Practice) 스타일입니다.

### [코드 분석]
- **핵심 로직**: `Pattern.splitAsStream()`을 통해 문자열을 스트림화하고, `map`에서 단어를 뒤집은 뒤 `joining(" ")`으로 결합합니다.
- **특징**:
    1. **공백 처리**: 연속된 공백(`"  "`)이 있어도 정규식(` +`)이 이를 하나의 구분자로 인식하여 유연하게 대응합니다.
    2. **결과 무결성**: V1과 달리 문장 끝에 불필요한 공백이 생기지 않습니다.

### [실무 활용 사례]
- **데이터 전처리(ETL)**: 로그 데이터나 사용자 입력값에서 단어의 순서는 유지하되, 특정 규칙에 따라 텍스트를 변환해야 하는 가공 단계에 적합합니다.
- **API 개발**: 가독성이 최우선인 Spring Boot 기반의 비즈니스 로직 작성 시 유지보수 비용을 낮추기 위해 사용됩니다.

---

## 3. reverse: 전체 문자열 역전
단어 단위가 아닌 데이터 전체의 순서를 바꾸는 헬퍼 메서드입니다.

### [코드 분석]
- **핵심 로직**: `StringBuilder(str).reverse()`를 호출합니다.
- **특징**: 내부적으로 `char[]` 배열의 인덱스를 직접 Swap(맞교환)하므로 성능이 압도적으로 빠릅니다.

### [실무 활용 사례]
- **회문(Palindrome) 검사**: 사용자가 입력한 단어(예: "토마토", "Level")가 거꾸로 해도 동일한지 판별하는 로직의 핵심 도구입니다.
- **데이터 보안 및 난독화**: 간단한 암호화 로직에서 바이트 순서를 뒤집거나, 통신 프로토콜의 엔디언(Endian) 변환 시 응용됩니다.
