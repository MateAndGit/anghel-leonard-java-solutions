# [Java] Strings 클래스 분석: 다중 문자열 결합 (Joining Strings)

이 예제는 여러 문자열을 구분자와 함께 연결하는 세 가지 세대별 방식을 보여줍니다. '마지막 구분자 처리'를 어떻게 우아하게 해결하는지가 관전 포인트입니다.

---

## 1. 메서드별 상세 분석

### ① V1: 전통적인 StringBuilder 방식
- **동작**: `for` 루프를 돌며 문자열과 구분자를 번갈아 `append`합니다.
- **특징**: 마지막 요소 뒤에는 구분자가 붙지 않도록 `args.length - 1`까지만 루프를 돌고 마지막에 별도로 추가하는 처리가 들어있습니다. (고전적이고 성능이 좋지만 코드가 깁니다.)

### ② V2: Java 8 Stream API 방식
- **동작**: 배열을 스트림으로 변환한 뒤 `Collectors.joining()`을 사용하여 합칩니다.
- **특징**: 가독성이 가장 좋으며, 내부적으로 구분자 처리를 완벽하게 해주므로 실무에서 가장 권장되는 방식입니다.

### ③ V3: StringJoiner 방식 (Java 8+)
- **동작**: 문자열 결합을 위해 탄생한 전용 클래스인 `StringJoiner`를 사용합니다.
- **특징**: 구분자뿐만 아니라 접두사(Prefix, `[`), 접미사(Suffix, `]`)도 쉽게 붙일 수 있는 유연함을 가집니다.

---

## 2. 실무 활용 사례

### [사례 1: CSV 데이터 생성]
데이터베이스에서 조회한 여러 컬럼 값을 쉼표(`,`)로 구분된 한 줄의 문자열로 만들어 파일로 내보낼 때 사용합니다.
- 예: `joinByDelimiterV2(',', "홍길동", "30", "서울")` -> `"홍길동,30,서울"`

### [사례 2: SQL IN 절 쿼리 생성]
DB 쿼리를 짤 때 `WHERE id IN (1, 2, 3)`과 같은 형식을 만들기 위해 여러 ID 값을 합칠 때 유용합니다.

### [사례 3: 로그(Log) 기록]
에러가 발생했을 때 여러 관련 변수값들을 한눈에 보기 좋게 `" | "` 같은 구분자로 합쳐서 로그 파일에 남길 때 활용합니다.

---

## 3. 실무형 요약 보고서

| 방식 | 가독성 | 특징 | 추천 상황 |
| :--- | :---: | :--- | :--- |
| **V1 (StringBuilder)** | 낮음 | 구현이 원초적임 | 아주 낮은 자바 버전 호환 필요 시 |
| **V2 (Stream)** | **최상** | 코드가 간결하고 현대적임 | **대부분의 실무 비즈니스 로직** |
| **V3 (StringJoiner)** | 상 | 결합 전용 클래스 활용 | 대괄호 `[]` 등을 함께 붙여야 할 때 |

---

## 4. 현업 개발자의 유지보수 팁 📝

### ⚠️ 가변 인자 (String... args) 활용
메서드 파라미터에 쓰인 `...`은 **가변 인자**라고 하며, 호출할 때 인자 개수를 마음대로 조절할 수 있게 해줍니다.
- `joinByDelimiterV2(',', "A", "B")` (O)
- `joinByDelimiterV2(',', "A", "B", "C", "D")` (O)



### 💡 알고 계셨나요? (Java 8+ 내장 메서드)
사실 자바 8부터는 `String.join()`이라는 메서드가 이미 내장되어 있습니다.
```java
// 실무에서 가장 많이 쓰는 실제 코드
String result = String.join(",", "Java", "Python", "C++");

```
